**入库时间：** 2026-05-12
**状态：** 🟢 活跃（已沉淀→AGENTS.md免请示清单）

**事件描述：**
需要在 Gateway 运行时清理 sessions.json 中的过期 session key。

**关键发现：**
Gateway 运行时禁止直接编辑 sessions.json，必须用 Python 脚本原子操作：
1. 读取 sessions.json
2. 修改 JSON（删除目标 key）
3. 写回 sessions.json
4. 删除对应的 .jsonl 文件
5. 重启网关（`openclaw gateway restart`）

必须包含重启网关步骤，否则 gateway 会在下次写入时覆盖回原状态。

**关联铁律/规则：**
- Session 清理操作 → AGENTS.md「按已沉淀安全流程执行，无需每次请示」清单

**入库时间：** 2026-05-12
**状态：** 🟡 已关闭（结论已形成，记录保留）

**事件描述：**
用户感知每天零点会话被重置，排查真正原因。

**关键发现：**
- 主会话（`agent:main:main`）**不存在**零点自动刷新
- 搜索范围：Gateway config schema、Cron jobs、Control UI JS bundle、全部文档

- `/session max-age off` 可彻底禁用会话最大寿命限制
